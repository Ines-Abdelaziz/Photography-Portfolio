# Ines Abdelaziz — Travel Photography

A personal photography portfolio showcasing shots from recent trips across Europe: Strasbourg, Paris, Luxembourg, Switzerland, Spain, and Cologne.

Still learning, still looking — this is a growing collection of frames from the road.

## Features

- Full-screen animated landing page
- Horizontal, draggable photo carousel for each destination
- Click-to-enlarge lightbox with keyboard navigation
- Scroll-triggered reveal animations
- Contact section
- No frameworks, no build step — plain HTML, CSS, and JavaScript

## Structure

```
index.html
images/<destination>/thumb/   grid-size photos (~700px)
images/<destination>/full/    lightbox-size photos (~1920px)
```

## Destinations

Strasbourg · Paris · Luxembourg · Switzerland · Spain · Cologne
